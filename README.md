# Breinbrekers · groep 8

Print-klare breinbreker-vellen voor groep 8, met eigen, originele puzzels.
Elk vel is één A4 met een "Naam:"-regel en 5 verschillende breinbrekers
(rekenpuzzel, getallenrij, logisch denken, woordpuzzel, codewoord, kraken,
raadsel en denk-en-teken). Achterin staat een antwoordblad voor de leerkracht.

## Bestanden
- `breinbrekers.html` — 5 vellen + antwoordblad, print-klaar op A4
- `generate.py` — generator; pas de lijst `PUZZELS` aan en draai opnieuw

## Zelf aanpassen
```bash
python3 generate.py
```

## Printen
Open `breinbrekers.html`, druk op Ctrl/Cmd + P, kies A4, marges "geen"/"standaard"
en vink "achtergrondafbeeldingen" aan voor de zachte kleuren.
